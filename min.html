API (Application Programming Interface) — це набір визначених правил, що дозволяють одному програмному продукту взаємодіяти з іншим. API виступає як посередник між програмами, забезпечуючи взаємодію між ними, не вимагаючи від розробників деталей реалізації іншої сторони.

alert - це функція (та їй подібні функції) якої немає в самому js, ця функція є в браузері.
