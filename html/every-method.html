The every method


const products = [
	{ name: "apple", quantity: 2 },
	{ name: "orange", quantity: 5 },
	{ name: "plum", quantity: 0 },
];

const hasEveryProduct = products.every(product => product.quantity > 0);
console.log(hasEveryProduct); // false

Масив об'єктів products

Це масив, який містить три об'єкти. Кожен об'єкт має дві властивості:

  • name — назва продукту (наприклад, "apple").
  • quantity — кількість продукту (наприклад, 2 для яблук, 5 для апельсинів, 0 для слив).

Метод .every()

Метод .every() перевіряє, чи всі елементи масиву відповідають певній умові. Він приймає колбек-функцію і повертає true, якщо кожен елемент масиву задовольняє умову, або false, якщо хоча б один елемент не відповідає умові.

Колбек-функція — це функція, яку передають в іншу функцію як аргумент. У нашому випадку, метод .every() вимагає, щоб йому передали функцію, яка буде виконана для кожного елемента масиву.

Ось ця частина product => product.quantity > 0 і є колбек-функцією. Вона передається як аргумент методу .every().

У данному випадку: для кожного продукту (об'єкта) у масиві products ми перевіряємо, чи кількість (quantity) більша за 0. 


const hasEveryProduct = products.every(function(product) {
	return product.quantity > 0;
});

hasEveryProduct буде false, оскільки не всі продукти мають кількість, більшу за 0.

Еквівалентна функція зі звичайним використанням return виглядала б так: 


function(product) {
	return product.quantity > 0;
}
