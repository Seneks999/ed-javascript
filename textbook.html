Введение в язык JavaScript для новичков


<script>
	alert('Привет, мир!');
</script>

Pазберем код программы. В нашем примере есть функция alert(), которая выводит текст на экран в виде диалогового окошка.

Функция состоит из имени (в нашем случае это alert) и круглых скобок, написанных после этого имени. В этих круглых скобках следует писать параметры функции. В нашем случае параметром является текст, который выводится на экран. В нашем случае функция имеет один параметр, однако бывают функции, в которые нужно передавать несколько параметров. В этом случае эти параметры пишутся через запятую.

Можно подключать не один, а несколько файлов с помощью нескольких тегов script.

В теге script можно либо писать код, либо подключить файл.

Браузер може кешувати підключені JavaScript файли, та в якийсь момент під час редагування коду браузер застосовуватиме попередню версію коду, а не поточну. Для боротьби з цім можна чистити кеш браузера, або під час підключення скрипта після імені файлу ставити знак питання, v, знак дорівнює, та число яка вказує номер версії ?v=1. Коли браузер кешує файл, вам просто потрібно буде збільшити номер на одиницю. 

<script src="script.js?v=1"></script>

Багаторядковий коментар в js

/* Перщий рядок, 
           Другий рядок ... */

Змінна (сновне поняття в будь-якій мови програмування) - це контейнер, у якому можема зберігати дані (рядки, числа). Кожна змінна має ім'я, яке складається з латинських літер, чисел, символів $ і знаків підкреслення (str, my_str, myStr, a1, $, $a, $$a). Але перший символ імені змінної не повинен бути цифрою.
